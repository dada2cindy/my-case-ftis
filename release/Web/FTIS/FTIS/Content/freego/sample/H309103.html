H309103 具體指出按下Tab鍵在表單控制項，超連結及物件間移動的順序

不良設計：

未使用tabindex屬性設定TAB鍵在表單控制項的移動順序。

不良設計：

  <TEXTAREA name="field1">輸入您第一題答案</TEXTAREA>
  <TEXTAREA name="field2">輸入您第二題答案</TEXTAREA>
  <INPUT type="submit" value="送出答案">
  <A href="#top">回到頁首</A>

範例結束

正確示範

使用tabindex屬性設定TAB鍵在表單控制項的移動順序，使用者按下TAB鍵時，控制鍵的順序將會依照連結中tabindex的設定值大小決定移動順序。

正確範例：

  <TEXTAREA name="field1" tabindex="2">輸入您第一題答案</TEXTAREA>
  <TEXTAREA name="field2" tabindex="3">輸入您第二題答案</TEXTAREA>
  <INPUT type="submit" tabindex="4" value="送出答案">
  <A href="#top" tabindex="1">回到頁首</A>

範例結束
