H203002 在doctype標籤中，使用標準規範的敘述以識別HTML版本類型

正確示範

各網頁應該在doctype標籤中，宣告該網頁所使用的HTML標準版本類型，以供使用者識別。例如網頁使用HTML 4.01版本語言撰寫時，可在W3C網站上相關網頁（http://www.w3.org/TR/html401/struct/global.jsp#idx-document_type_declaration-3）查詢到正確的寫法。而所謂的DTD就是「文件類型定義」（document type definition）的縮寫。

下面是宣告HTML 4.01版本語言的三種寫法：

正確範例：

 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
 
 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

範例結束
