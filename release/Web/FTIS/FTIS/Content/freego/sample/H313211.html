H313211 在網頁標題、段落、及列表之前，提供辨別訊息以利識別

不良設計：

下面例子並未提供資訊的辨別訊息，因此在未讀取資訊內容前，使用者無法知道下面的四條訊息是天才書局的最新消息。

不良設計：

  <UL>
  <LI>最新消息一</LI>
  <LI>最新消息二</LI>
  <LI>最新消息三</LI>
  <LI>最新消息四</LI>
  </UL>

範例結束

正確示範

使用標題標籤在最新消息之前提供可辨別的資訊，使用者可以知道標題下面的四條訊息是天才書局的最新消息。

正確範例：

  <H1>天才書局最新消息</H1>
  <UL>
  <LI>最新消息一</LI>
  <LI>最新消息二</LI>
  <LI>最新消息三</LI>
  <LI>最新消息四</LI>
  </UL>

範例結束
