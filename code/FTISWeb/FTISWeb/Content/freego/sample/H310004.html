H310004 在網頁文字輸入區中須有預設值

不良設計：

在文字輸入區中，沒有預設的文字提示。

不良設計：

  <FORM>
  <LABEL for="username1">姓名：</LABEL>
  <INPUT type="text" name="yourname" id="username1">
  <LABEL for="jobdesc1">工作經歷：</LABEL>
  <TEXTAREA name="jobdescription" id="jobdesc1" rows="5" cols="60">
  </TEXTAREA>
  <INPUT type="submit" value="送出資料">
  <INPUT type="reset" value="重新輸入">
  </FORM>

範例結束

正確示範

在文字輸入區中，提示文字讓使用者知道該文字輸入區需輸入何種資訊。

正確範例：

  <FORM>
  <LABEL for="username2">姓名：</LABEL>
  <INPUT type="text" name="yourname" id="username2" value="輸入您的姓名">
  <LABEL for="jobdesc2">工作經歷：</LABEL>
  <TEXTAREA name="jobdescription" id="jobdesc2" rows="5" cols="60">
  -- 請輸入您的工作經歷--
  </TEXTAREA>
  <INPUT type="submit" value="送出資料">
  <INPUT type="reset" value="重新輸入">
  </FORM>

範例結束
