H205105 資料表格須提供標題說明

不良設計：

沒有使用CAPTION標籤提供表格標題說明，使用者無法在第一時間了解表格中提供的資料。

不良設計：

  <TABLE border="1" summary=".....(摘要說明)....">
  <TR>
  <TH>主要城市</TH>
  <TH>7月12日</TH>
  <TH>7月13日</TH>
  </TR>
  <TR>
  <TD>台北</TD>
  <TD>多雲</TD>
  <TD>雷雨</TD>
  </TR>
  <TR>
  <TD>台中</TD>
  <TD>多雲</TD>
  <TD>多雲</TD>
  </TR>
  <TR>
  <TD>高雄</TD>
  <TD>多雲</TD>
  <TD>陣雨</TD>
  </TR>
  </TABLE>

範例結束

正確示範

提供表格標題說明，使用者可以在第一時間了解表格所提供的資訊。

正確範例：

  <TABLE border="1" summary=".....(摘要說明)....">
  <CAPTION>
  台灣主要城市週末天氣概況
  </CAPTION>
  <TR>
  <TH>主要城市</TH>
  <TH>7月12日</TH>
  <TH>7月13日</TH>
  </TR>
  <TR>
  <TD>台北</TD>
  <TD>多雲</TD>
  <TD>雷雨</TD>
  </TR>
  <TR>
  <TD>台中</TD>
  <TD>晴天</TD>
  <TD>多雲</TD>
  </TR>
  <TR>
  <TD>高雄</TD>
  <TD>晴天</TD>
  <TD>晴天</TD>
  </TR>
  </TABLE>

範例結束
