H310005 勿單以空白間隔分開相連之超連結

不良設計：

連結與連結之間使用空白鍵、使用表格的TD、使用&nbsp;、使用span分開連結。

不良設計：

  <P>
  <A href="http://www.yam.com/">蕃薯藤</A> 
  <A href="http://www.pchome.com.tw/">PChome</A> 
  <A href="http://www.yahoo.com.tw/">Yahoo!奇摩</A>
  </P>
  <TABLE>
    <TR>
      <TD><A href="http://www.ey.gov.tw/">行政院</A></TD>
      <TD><A href="http://www.ly.gov.tw/index.jsp">立法院</A></TD>
      <TD><A href="http://www.judicial.gov.tw/">司法院</A></TD>
    </TR>
  </TABLE>
  <P>
  <A href="http://www.president.gov.tw/">總統府</A> &nbsp;
  <A href="http://www.exam.gov.tw/">考試院</A> &nbsp;
  <A href="http://www.cy.gov.tw/">監察院</A>
  </P>
  <P>
  <A href="http://www.ntu.edu.tw/">台灣大學</A><span>
  <A href="http://www.nthu.edu.tw/">清華大學</A><span>
  <A href="http://www.tku.edu.tw/">淡江大學</A>
  </P>

範例結束

正確示範

連結與連結之間加上[ ]、( )、{ }、|、－…等符號加以區隔。

正確範例：

[<A href="http://www.yam.com/">蕃薯藤</A>] 
[<A href="http://www.pchome.com.tw/">PChome</A>]
[<A href="http://www.yahoo.com.tw/">Yahoo!奇摩</A>]
