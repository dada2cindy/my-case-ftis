H213004 為你的網頁加上標題

不良設計：

下面是一個不良的設計，沒有為網頁設定名稱，所以在瀏覽器的左上方不會出現網頁的名稱。

不良設計：

  <HEAD>
  <META http-equiv="Content-Type" content="text/html; charset=big5">
  </HEAD>

範例結束

正確示範

使用TITLE標籤替網頁設定標題，所以在瀏覽器的左上方會出現網頁的名稱。

正確範例：

  <HEAD>
  <TITLE>無障礙網頁空間發展小組</TITLE>
  <META http-equiv="Content-Type" content="text/html; charset=big5">
  </HEAD>

範例結束
