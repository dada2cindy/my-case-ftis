H203107 項目符號及編號之標籤(如li、ul)僅可使用於實際網頁內容的項目條列，不可用於編輯格式

不良設計：

<H3>台灣在無障礙網路的推動</H3>
<UL>
我國政府在1997年公佈「身心障礙保護法」，並開始推動在公共場所設置無障礙空間。
台灣地區在這五、六年第一波的努力之下已經有相當成效，
大部份的公共場所都有設置無障礙坡道、愛心鈴、殘障車位等無障礙空間。
</UL>

範例結束
