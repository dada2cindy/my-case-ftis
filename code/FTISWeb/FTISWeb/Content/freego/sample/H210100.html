H210100 除非使用者知道將會開啟一個新視窗，不要隨便開啟一個新視窗

不良設計：

沒有告知使用者將開啟一個新的視窗

不良設計：

  <A href="http://www.cpbl.com.tw/" target="_blank">中華職棒全球資訊網</A>

範例結束

正確範例：

利用文字說明或是其他方式告知使用者將開啟一個新的視窗

正確範例：

   
<P>國內外相關連結<br>
<UL> 
<LI><A href="http://www.yahoo.com" target="_blank">YAHOO奇摩(另開新視窗)</A></LI> 
<LI><A href="http://www.google.com" target="_blank">Google(另開新視窗)</A></LI> 
</UL>
</P>

範例結束
