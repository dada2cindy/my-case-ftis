H213202 指向不同網址的超連結，不可使用相同的超連結說明

不良設計：

  <A href="http://www.gov.tw">首頁</A>&gt;<a href="ex.jsp">行政院</a>
  &gt;<a href="ex.jsp">行政院研考會</a>&gt;最新消息
  <H2>最新消息...</H2>
  <P>最新消息內容...</P>
  相關消息請至行政院研考會網站<A href="http://www.rdec.gov.tw">首頁</A>

範例結束

正確示範

在A標籤中使用title屬性來區別彼此之間的不同。

正確範例：

  <A href="http://www.gov.tw" title="回我的E化政府網站首頁">首頁</A>
  &gt;<A href="ex.jsp">行政院</A>&gt;<A href="ex.jsp">行政院研考會</A>&gt;最新消息
  <H2>最新消息...</H2>
  <P>最新消息內容...</P>
  相關消息.....請至行政院研考會網站
  <A href="http://www.rdec.gov.tw" title="回行政院研考會首頁">首頁</A>

範例結束

  <A href="http://www.gov.tw">首頁</A>
