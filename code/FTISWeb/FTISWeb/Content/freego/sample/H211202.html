H211202 避免使用過時的HTML語法

不良設計：

使用過時的DIR標籤，來達到目錄清單的效果，HTML4.0已將此標籤自標準規範中移除 ，若要了解有哪些標籤語言已被移除，可至W3C網站查詢。

不良設計：

  <P>手機品牌銷售排行榜</P>
  <DIR>
  <LI>三星</LI>
  <LI>夏普</LI>
  <LI>諾基亞</LI>
  <LI>明碁</LI>
  </DIR>

範例結束

正確示範

使用HTML4.0中仍提供的UL標籤來達到相同目錄清單的效果。

正確範例：

  <P>手機品牌銷售排行榜</P>
  <UL>
  <LI>三星</LI>
  <LI>夏普</LI>
  <LI>諾基亞</LI>
  <LI>明碁</LI>
  </UL>

範例結束
